Step 1
Query Reception
User sends a message via chat UI or POST /support. The API receives the raw text.
Step 2
Intent Classification
The Intent Classifier Agent calls the LLM to categorise the query into billing, technical, account, general, or complaint.
Step 3
Knowledge Retrieval
The Knowledge Retriever Agent searches the SQLite knowledge base for articles matching the detected intent and keywords.
Step 4
Response Generation
The LLM synthesises retrieved articles into a clear, contextual answer returned to the user with confidence score.
