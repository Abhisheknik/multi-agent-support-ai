Step 1
Query Reception
User sends a message via chat UI or POST /support. The API receives the raw text.
Step 2
Intent Classification
The Intent Classifier Agent calls Gemini to categorise the query into billing, technical, account, general, or complaint.
Step 3
Knowledge Retrieval
The Knowledge Retriever Agent searches the SQLite knowledge base for articles that match the detected intent and keywords.
Step 4
Response Generation
Gemini 2.5 Flash synthesises the retrieved articles into a clear, contextual answer returned to the user.
